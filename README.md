# ✊✋✌️ Rock-Paper-Scissors

This is a simple Rock-Paper-Scissors game built using JavaScript as part of **The Odin Project** JavaScript curriculum.

🕹️ **Play the game here:**  
👉 [Rock-Paper-Scissors Game](https://idanshiju.github.io/Rock-Paper-Scissors/)

---

## 📖 Description

This web-based game allows you to play Rock-Paper-Scissors against the computer. It's designed to demonstrate core JavaScript concepts such as functions, conditional statements, and DOM interaction.

> ⚠️ **Note:** The game displays results using the **Developer Console**.

---

## 🔍 How to View the Result

To see the results while playing, open **Chrome DevTools** and navigate to the **Console** tab. Here are three ways to open DevTools:

1. **Chrome Menu:**  
   `More Tools` → `Developer Tools`

2. **Right-Click Method:**  
   Right-click anywhere on the page → Select `Inspect`

3. **Keyboard Shortcut:**  
   - Windows: `F12` or `Ctrl + Shift + I`  
   - Mac: `Opt + Cmd + I`

---

## 🔁 Troubleshooting

If the game doesn't display the result after clicking a button:

- Click **Cancel** if prompted.
- Refresh the page and try again.
